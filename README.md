# 臥龍一號 (Wolong Oracle)

A deterministic oracle system inspired by classical Chinese divination texts such as I Ching (易經) and Zhuge Shenshu (諸葛神數).

---

## 🧭 What is this?

This project explores a simple idea:

> Can structured symbolic systems create meaningful interpretations of human questions?

Instead of using AI or probabilistic models, this system uses a **fixed 384-state mapping** to convert user input into a symbolic response.

---

## ⚙️ How it works

1. You enter a question (any natural language input)
2. The system converts the input into a deterministic index (1–384)
3. Each index maps to a predefined oracle entry
4. The output is a short poetic / symbolic interpretation

There is:

- No AI
- No LLM
- No randomness (beyond deterministic hashing)

---

## 📦 Structure (WIP)

- 384 fixed oracle entries
- Input → deterministic mapping function
- Output → symbolic verse + interpretation

---

## 🧠 Design philosophy

This project is not about prediction accuracy.

It is closer to:

- Symbolic reflection systems
- Structured ambiguity interpretation
- Cognitive projection tools

The goal is to observe:

> how humans extract meaning from constrained symbolic outputs.

---

## ⚠️ Disclaimer

This is not a predictive system.

It is an experimental symbolic mapping system for exploration and entertainment purposes.
